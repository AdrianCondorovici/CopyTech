# CopyTech
Proiect personal pentru exersarea notiunilor de CSS/EJS/JS plus realizarea unei baze de date folosind mongoose si a unui sitem de autentificare folosind passport

Proiectul a avut ca scop inglobarea mai multor biblioteci intalnite in cursurile si tutorialele studiate intr-o singura entitate anume un site web.

Cateva dintre bibliotecile folosite sunt:
-node.js
-mongoose
-flash
-express
-passport
-session
